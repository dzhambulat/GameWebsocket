Запуск бекенда 

`
cd backend
npm install
npm start
`

Запуск фронта

`
cd frontend
npm install
npm run dev
`

после этого открываете адрес в двух табах браузера.
В одном табе нажимаете create game. берете id игры и указываете в другом табе и Join.

Возможные улучшения:
Использовать Redis в случае нескольких экземляров сервера для sticky sessions
Хранить игру в Redis, и фоновым процессам сохранять ее в PostgresSQL если нужна история игр
